## Help와 문서 활용하기

```
git help
```

> 기본적인 명령어들과 설명<br />

<br />

```
git help -a
```

> Git의 모든 명령어들<br /> > `j`로 내리기, `k`로 올리기, `:q`로 닫기<br />

<br />

```
git (명령어) -h
```

> 해당 명령어의 설명과 옵션 보기

<br />

```
git help (명령어) -w # (추가 설치 필요함)
```

```
git (명령어) --help
```

> 해당 명령어의 설명과 옵션 웹사이트에서 보기
